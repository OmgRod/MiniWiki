# Introduction

MiniWiki is a lightweight documentation template powered by **Next.js**, **MDX**, and **TailwindCSS**.

## What MiniWiki is for

- Internal team docs
- Product knowledge bases
- Technical runbooks
- Public docs portals

## How this documentation is organized

- **Getting Started**: initial setup and structure
- **Configuration**: JSON-driven navigation and site behavior
- **Authoring**: writing content with MDX, components, and icons
- **Theming**: site-wide style presets
- **Operations**: publishing flow and troubleshooting

# Quickstart

## Requirements

- Node.js 18+
- npm 9+

## Install

```bash
npm install
```

## Run development server

```bash
npm run dev
```

Open `http://localhost:3000`.

## Build production output

```bash
npm run build
npm run start
```

<Alert type="success" title="You are ready">
MiniWiki is now running locally.
</Alert>

# Quickstart

## Requirements

- [Node.js](https://nodejs.org/) 18+ (LTS recommended)
- [npm](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm) 9+
- [Git](https://git-scm.com/downloads)

## Verify prerequisites

```bash
node --version
npm --version
git --version
```

## Clone MiniWiki

```bash
git clone https://github.com/OmgRod/MiniWiki.git
cd MiniWiki
```

## Install

```bash
npm install
```

## Run development server

```bash
npm run dev
```

Open `http://localhost:3000`.

## Build production output

```bash
npm run build
npm run start
```

<Alert type="success" title="You are ready">

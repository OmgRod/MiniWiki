## Icon system

Icons use `lucide-react` through the `Icon` component.

```mdx
<Icon name="settings" />
<Callout icon="rocket">Deploy docs faster.</Callout>
```

<Callout icon="idea">
Prefer icon names over emojis for a clean, consistent UI.
</Callout>

## Common built-ins

- `Alert`, `Callout`, `Badge`
- `Card`, `CardGrid`, `Divider`
- `CodeBlock`, `ButtonLink`, `Kbd`
- `Accordion`, `TabPanels`, `Steps`
- `Icon`

# Content Structure

MiniWiki generates routes from MDX files under `content/`.

## Route mapping rules

- `content/home.mdx` → `/`
- `content/guides/quickstart.mdx` → `/guides/quickstart`
- `content/guides/site-config.mdx` → `/guides/site-config`

## Recommended structure

```text
content/
  home.mdx
  guides/
    introduction.mdx
    quickstart.mdx
    ...
  miniwiki.config.json
  site.json
  templates.json
```

<Callout icon="layers">
Keep docs pages inside `content/guides/` for a clean and predictable wiki structure.
</Callout>

- All navigation and site settings are now part of `miniwiki.config.json`.

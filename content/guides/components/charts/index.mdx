# Charts Index

Chart-related components are grouped under `components/charts/` instead of the base root component set.

## When to use which chart

- Use `BarChart` or `StackedBarChart` for category comparisons.
- Use `SparklineChart` or `AreaChart` for trends over time.
- Use `DonutChart` and `ProgressRingChart` for composition/KPI snapshots.
- Use `HeatmapChart` for matrix-style intensity views.

## Data notes

- All chart components accept direct `data` arrays from MDX.
- Components include safe fallback demo data when empty so docs previews stay visible.

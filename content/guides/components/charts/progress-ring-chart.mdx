# ProgressRingChart

## Props

| Prop | Type | Required | Description |
| --- | --- | --- | --- |
| `value` | `number` | Yes | Progress percent (0-100). |
| `size` | `number` | No | Ring size. Default: `180`. |
| `strokeWidth` | `number` | No | Ring thickness. Default: `14`. |
| `label` | `string` | No | Text label below ring. |
| `className` | `string` | No | Wrapper classes. |

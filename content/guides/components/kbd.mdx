# Kbd

Display keyboard shortcuts in docs.

## Props

| Prop | Type | Required | Description |
| --- | --- | --- | --- |
| `children` | `ReactNode` | Yes | Key text/content. |

## Live examples

Press <Kbd>Ctrl</Kbd> + <Kbd>K</Kbd> to focus search.

Use <Kbd>Ctrl</Kbd> + <Kbd>S</Kbd> to save your current edit.

## Code

```mdx
Press <Kbd>Ctrl</Kbd> + <Kbd>K</Kbd> to focus search.
```

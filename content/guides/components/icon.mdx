# Icon

Render icons by name for consistent visual language.

## Props

| Prop | Type | Required | Description |
| --- | --- | --- | --- |
| `name` | `string` | Yes | Icon key (e.g., `settings`, `rocket`). |
| `size` | `number` | No | Icon size in pixels. |
| `className` | `string` | No | Extra utility classes. |

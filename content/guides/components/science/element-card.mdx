# ElementCard

## Props

| Prop | Type | Required | Description |
| --- | --- | --- | --- |
| `symbol` | `string` | Yes | Element symbol. |
| `name` | `string` | Yes | Element name. |
| `number` | `number` | Yes | Atomic number. |
| `group` | `string` | No | Category/group label. |
| `atomicMass` | `string \| number` | No | Atomic mass value. |
| `summary` | `string` | No | Supporting description text. |
| `className` | `string` | No | Wrapper classes. |

<ElementCard

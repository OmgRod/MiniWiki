# Infobox

Use `Infobox` for article metadata summaries.

## Props

| Prop | Type | Required | Description |
| --- | --- | --- | --- |
| `title` | `string` | No | Infobox heading. |
| `image` | `string` | No | Image URL/path. |
| `imageAlt` | `string` | No | Image alt text. |
| `imageCaption` | `string` | No | Caption under image. |
| `items` | `Array<{ label: string, value: ReactNode }>` | Yes | Key-value metadata rows. |

## Live examples

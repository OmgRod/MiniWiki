# Callout

Use callouts for important narrative notes and tips.

## Props

| Prop | Type | Required | Description |
| --- | --- | --- | --- |
| `icon` | `string` | No | Icon name resolved by `Icon`. |
| `variant` | `'default' \| 'caution' \| 'error' \| 'note' \| 'success'` | No | Color variant for the callout block. |
| `children` | `ReactNode` | Yes | Callout content. |

## Examples

### Live examples

<Callout>
  Keep docs task-oriented and skimmable.
</Callout>

<Callout variant="caution" icon="alert">
  This action can affect existing navigation and links.
</Callout>

<Callout variant="error" icon="error">
  Build failed because a referenced page path does not exist.
</Callout>

<Callout variant="note" icon="book">
  This is a note-style callout for context and reference details.
</Callout>

<Callout variant="success" icon="check">
  Configuration saved and applied successfully.
</Callout>

<Callout icon="rocket">
  Push to main to publish to GitHub Pages.
</Callout>

<Callout icon="tool">
  Update the `sidebar` section in `miniwiki.config.json` whenever you add new guides.
</Callout>

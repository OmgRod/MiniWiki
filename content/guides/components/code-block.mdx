# CodeBlock

`CodeBlock` is used directly and as the default renderer for `pre` blocks.

## Behavior

- Detects code language from fenced code blocks.
- Shows copy button in rendered blocks.
- Supports light and dark themes.

## Direct example

```tsx
const isReady = true;

if (isReady) {
	console.log('MiniWiki is ready');
}
```

## Live markdown render example

```js
export function add(a, b) {
	return a + b;
}
```

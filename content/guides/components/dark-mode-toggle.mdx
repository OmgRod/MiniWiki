# DarkModeToggle

Provides the light/dark toggle button used in the header.

## Behavior

- Toggles document theme class.
- Persists preference for future visits.
- Exposed as a layout/navigation component (not usually embedded in MDX pages).

	<DarkModeToggle />
</div>

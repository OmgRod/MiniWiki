# Math Index

Math components are grouped under `components/math/`.

## Practical guidance

- Use `MathEquation` for single formulas.
- Use `FormulaList` for reference-style pages.
- Use `MathSteps` for derivations and tutorials.
- Use `UnitConverter` for interactive STEM explainers.

- [MathEquation](/guides/components/math/math-equation)
- [FormulaList](/guides/components/math/formula-list)

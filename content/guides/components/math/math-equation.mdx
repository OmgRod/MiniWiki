# MathEquation

Renders LaTeX expressions through KaTeX for technical docs, tutorials, and science posts.

## Props

| Prop | Type | Required | Description |
| --- | --- | --- | --- |
| `expression` | `string` | Yes | LaTeX expression to render. |
| `block` | `boolean` | No | Block mode when true, inline mode when false. Default: `true`. |
| `className` | `string` | No | Wrapper classes. |

## Live examples

# Card

Build visual content blocks for summaries and navigation hubs.

## Props

| Prop | Type | Required | Description |
| --- | --- | --- | --- |
| `title` | `string` | No | Card title. |
| `description` | `string` | No | Supporting summary. |
| `children` | `ReactNode` | No | Additional body content. |

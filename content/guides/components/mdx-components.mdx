# mdx-components Registry

File: `components/mdx-components.js`

This registry maps imported React components to MDX tags.

## Responsibilities

- Exposes components like `Alert`, `Card`, `Infobox`, and `WikiTable` primitives.
- Overrides markdown element renderers (`pre`, `table`, `thead`, `tbody`, `tr`, `th`, `td`, `caption`).
- Defines what can be used directly inside `.mdx` content.

## When to update

- After adding a new component intended for MDX pages.
- When changing global markdown element rendering behavior.

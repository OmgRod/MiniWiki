# Figure

`Figure` wraps media with consistent border/caption styling.

## Props

| Prop | Type | Required | Description |
| --- | --- | --- | --- |
| `src` | `string` | Yes | Image source URL/path. |
| `alt` | `string` | No | Accessible alt text. |
| `caption` | `string` | No | Figure caption text. |
| `width` | `string` | No | CSS width value. |

# JSON Reference Index

Every key in every JSON config file is documented in this section.

## Which file to edit

| Goal | File |
| --- | --- |
| Change site name, default preset, edit-page behavior | `content/site.json` |
| Change top navigation links | `content/header.json` |
| Change sidebar structure and ordering | `content/sidebar.json` |
| Change page-template defaults and registry | `content/templates.json` |
| Change footer columns, links, and copyright | `content/footer.json` |

## Fast path

1. Pick the file from the table above.
2. Update the key you need.
3. Save and refresh the site.

- [site.json](/guides/json-reference/site-json)
- [header.json](/guides/json-reference/header-json)
- [sidebar.json](/guides/json-reference/sidebar-json)
- [templates.json](/guides/json-reference/templates-json)
- [footer.json](/guides/json-reference/footer-json)

# sidebar.json Reference

File: `content/sidebar.json`

## Most common edits

- Add a new sidebar section
- Add links under an existing section
- Reorder sections and links
- Toggle section collapse behavior

## Top-level keys

| Key | Type | Required | Description |
| --- | --- | --- | --- |
| `sections` | `array` | Yes | Ordered sidebar section list. |

## `sections[]` object keys

| Key path | Type | Required | Description |
| --- | --- | --- | --- |
| `sections[].title` | `string` | Yes | Section heading text. |
| `sections[].collapsible` | `boolean` | Yes | Whether section can be collapsed in UI. |
| `sections[].items` | `array` | Yes | List of navigable pages in section. |

## `sections[].items[]` object keys

| Key path | Type | Required | Description |
| --- | --- | --- | --- |
| `sections[].items[].title` | `string` | Yes | Navigation label for page. |
| `sections[].items[].path` | `string` | Yes | URL/path target for that item. |

## Notes

- Order of `sections` controls sidebar section order.
- Order of `items` controls link order within each section.

## Add a new section (copy-ready)

```json
{
	"title": "My Section",
	"collapsible": true,
	"items": [
		{ "title": "Overview", "path": "/guides/my-section/overview" },
		{ "title": "Reference", "path": "/guides/my-section/reference" }
	]
}
```

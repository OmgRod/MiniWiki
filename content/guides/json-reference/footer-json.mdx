# Footer Configuration

The footer configuration is now part of the `footer` section in `miniwiki.config.json`.

# footer.json Reference

File: `content/footer.json`

## Most common edits

- Change footer columns and link groups
- Update legal links in `bottomLinks`
- Use dynamic year in `copyright`

## Top-level keys

| Key | Type | Required | Description |
| --- | --- | --- | --- |
| `columns` | `array` | Yes | Footer column definitions. |
| `tagLine` | `string` | Yes | Footer tagline text. |
| `bottomLinks` | `array` | Yes | Lower-row legal/utility links. |
| `copyright` | `string` | Yes | Copyright line. Supports `{{year}}` placeholder for current year. |

## `columns[]` object keys

| Key path | Type | Required | Description |
| --- | --- | --- | --- |
| `columns[].title` | `string` | Yes | Column heading text. |
| `columns[].links` | `array` | Yes | Link list within column. |

## `columns[].links[]` object keys

| Key path | Type | Required | Description |
| --- | --- | --- | --- |
| `columns[].links[].label` | `string` | Yes | Link display label. |
| `columns[].links[].url` | `string` | Yes | Destination URL/path/mailto. |

## `bottomLinks[]` object keys

| Key path | Type | Required | Description |
| --- | --- | --- | --- |
| `bottomLinks[].label` | `string` | Yes | Link display label. |
| `bottomLinks[].url` | `string` | Yes | Destination URL/path. |

## Auto year placeholder

Use `{{year}}` in the copyright text to automatically render the current year.

```json
{
	"copyright": "© {{year}} MiniWiki. All rights reserved."
}
```

## Footer link object (copy-ready)

```json
{ "label": "GitHub", "url": "https://github.com/OmgRod/MiniWiki" }
```

# Navigation JSON

MiniWiki navigation is fully config-driven.

## Sidebar

File: `content/sidebar.json`

- Define sections
- Add page links
- Set `collapsible` per section

## Header

File: `content/header.json`

- Define top nav links
- Keep links concise and task-oriented

## Practical advice

- Use sidebar for full structure
- Use header for key entry points
- Keep naming short and clear

<Callout icon="settings">
Changing JSON is enough — no code edits needed for nav updates.
</Callout>
